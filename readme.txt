WP Custom Admin Login Lite is a free WP login page customization plugin that can be used to customize your default login page with the WordPress Customizer options. Packaged with 5 beautifully pre-designed templates, the plugin offers lots of customization possibilities. With no coding skill, you can create an outstanding login page in no time. This fully customizer based plugin comes with intuitive interface that allows you to configure your admin login page as per your preference.

The plugin supports color/image background. WP Custom Admin Login Lite is an ultimate plugin to customize your admin login page in a unique and stunning way.

== Frequently Asked Questions ==
= What does this plugin do? =

This plugin lets you add your pre-built templates to your existing WP Login Page.

= Does the plugin work with Multi-site? =

You will be able to register the settings but, you will not be able to view changes through the Cusotmizer. This is due to current Customizer API limitation.

= I do not see immediate changes in the Customizer Preview? =

In a traditional sense, plugins are not support to work with WordPress Customizer, we use a hack to do so. If you do not find any immediate changes, please save the settings and refresh the page.

= Do I need to now any coding to use this plugin? =

If you just want a quick template in your site, you do not require any coding. But can add additional CSS if you want.

== Installation ==
1. Unzip [plugin].zip
2. Upload all the files to the /wp-content/plugins/wp-custom-admin-login-lite
3. Activate the plugin through the \'Plugins\' menu in WordPress.
4. For customizing the plugin settings, click on WP Custom Admin Login Lite option menu in WordPress left admin menu.
